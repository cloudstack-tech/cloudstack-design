# CloudStack Design System

欢迎来到 **CloudStack Design** 组件库！

这是一个专为云计算平台设计的现代化 UI 组件库，提供一套完整的设计系统和可复用的 React 组件。

## ✨ 特性

- 🎨 **现代化设计** - 遵循最新的 UI/UX 设计规范
- 🧩 **组件化** - 高度可复用的组件架构
- 🎯 **类型安全** - 完整的 TypeScript 支持
- 🌈 **主题系统** - 灵活的主题定制能力
- ♿ **无障碍访问** - 符合 WCAG 标准
- 📱 **响应式** - 支持各种屏幕尺寸
- ⚡ **高性能** - 优化的渲染性能

## 🚀 快速开始

### 安装

```bash
pnpm add @cloudstack-design/react @cloudstack-design/theme
```

### 使用

```tsx
import {Button} from "@cloudstack-design/react";

function App() {
  return (
    <Button color="primary" variant="solid">
      点击我
    </Button>
  );
}
```

## 📦 包结构

- **@cloudstack-design/react** - React 组件库
- **@cloudstack-design/theme** - 主题系统和样式
- **@cloudstack-design/system** - 样式系统核心
- **@cloudstack-design/system-rsc** - React Server Components 支持

## 🎯 设计原则

1. **简洁明了** - 组件 API 设计简单直观
2. **灵活可定制** - 提供丰富的定制选项
3. **性能优先** - 注重运行时性能和包体积
4. **开发体验** - 优秀的 TypeScript 类型提示和文档
